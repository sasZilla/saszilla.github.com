Регулярные выражение в JavaScript работают через специальный объект созданный классом RegExp. У объекта есть методы test,exec,search,match,replace.

Объект RegExp

  var req = new RegExp("b+", "i");
или
  var req = /b+/i;
Флаги - строка из любой комбинации символов g(глобальный поиск), i(регистр неважен) и m(многострочный поиск).
